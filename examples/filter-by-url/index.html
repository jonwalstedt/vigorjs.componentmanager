Filter components by url

  • Passing arguments ex: passing-arguments/:type/:id
  • Passing arguments ex: passing-arguments/:type/:id
  • Splats ex: splat/*path
  • Optional Arguments 1 ex: optional/:section(/:subsection)
  • Optional Arguments 2 ex: optional/:section(/:subsection)
  • Optional Arguments 3 ex: optional/:section(/:subsection)(/:id)
  • Global ex: all urls

You can filter components by specify a url pattern for your instance definition and then add the url you want to filter on to the filter passed to the refresh method.

You can add any of the url patterns that Backbone supports as url pattern on your instance definition. The component manager will filter out components that matches the url passed to the filter and instantiate them with url params passed to the constructor function.
