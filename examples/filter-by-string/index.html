Filter components by string matching

  • includeIfStringMatches
  • hasToMatchString
  • cantMatchString

There are three different default ways of filtering on a string:

  • Either the filter will include the component if the string matches but also include all components that doesnt have a filter string
  • Or it will return only components matching the string passed to the filter
  • Or it will return only components NOT matching the string passed to the filter

Try passing the string "test" to the three different methods of filtering by clicking the links above.
